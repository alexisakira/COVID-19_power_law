Replication files for "On the Emergence of a Power Law in the Distribution of COVID-19 Cases"
Physica D 412:132649 (2020)
https://doi.org/10.1016/j.physd.2020.132649
by Brendan K. Beare and Alexis Akira Toda

Confirmed to run with Matlab 2019b

Main file:
	- randgrowth_covid.m

Other files and folders:
	- ClausetShaliziNewman: codes for estimating Pareto distribution
	- cleanData.m: data cleaning file
	- discreteLogistic_ML.m: maximum likelihood estimation of discrete logistic
	- discreteLogistic_obj.m: objective function of maximum likelihood
	- note_discreteLogistic.pdf: note on discrete logistic distribution
